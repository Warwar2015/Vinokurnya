# Винокурня — Distiller's Assistant

Профессиональный инструмент для самогоноварения и дистилляции. Приложение включает инженерные калькуляторы, базу рецептов и систему заметок.

## ✨ Возможности

### 🧮 Инженерные калькуляторы

1. **Разбавление спирта с учетом температуры**
   - Расчет объема воды для разбавления
   - Коррекция по температуре смеси
   - Табличные данные плотности спирта

2. **Дробная перегонка**
   - Определение объемов фракций (головы, тело, хвосты)
   - Рекомендации по отбору
   - Оценка крепости тела

3. **Оптимальная скорость отбора**
   - Расчет капель в секунду
   - Интенсивность отбора
   - Рекомендации по оборудованию

4. **Температурные режимы**
   - Расчет температуры кипения воды
   - Коррекция по атмосферному давлению
   - Оптимальная температура перегонки

### 📖 Рецепты

- **Виски**: Классические односолодовые, торфяные, бюджетные
- **Бурбон**: Классический, ржаной, кукурузно-пшеничный
- **Настойки**: Лимонная, клюквенная, калгановка и другие

**Функции:**
- Поиск и фильтрация по категориям
- Добавление в избранное
- Рейтинг и отзывы
- Информация о времени и сложности

### 📝 Заметки

- Создание, редактирование, удаление заметок
- Категоризация по этапам: Брага, Перегонка, Выдержка, Дегустация
- Поиск по содержимому
- Отметка как любимые
- Экспорт/импорт в JSON

### 📊 Статистика

- Общая статистика по рецептам и заметкам
- Рецепты в избранном
- Средний рейтинг
- Популярные категории

### 🎨 Интерфейс

- Темная и светлая тема
- Интуитивный интерфейс
- Адаптивный дизайн

## 🚀 Установка и запуск

### Требования

- .NET 8.0 SDK
- Windows 10/11 (или выше)

### Установка

1. Клонируйте репозиторий:
```bash
git clone <repository-url>
cd VinokurnyaWpf
```

2. Восстановите пакеты:
```bash
dotnet restore
```

### Запуск

**Вариант 1: Через командную строку**
```bash
dotnet run
```

**Вариант 2: Через Visual Studio**
1. Откройте файл `VinokurnyaWpf.csproj`
2. Нажмите кнопку "Запустить" (Play)

**Вариант 3: Через Visual Studio Code**
1. Установите расширение .NET Install Tool
2. Откройте проект
3. Нажмите F5 для запуска

## 📖 Использование калькуляторов

### Разбавление спирта

1. Выберите вкладку "Калькуляторы"
2. Нажмите на кнопку "💧 Разбавление спирта"
3. Введите:
   - Объем спирта (мл)
   - Текущая крепость (%)
   - Целевая крепость (%)
   - Температура смеси (°C)
4. Нажмите "Рассчитать"
5. Результат отобразится ниже

### Дробная перегонка

1. Нажмите на кнопку "🔥 Дробная перегонка"
2. Введите:
   - Объем сырца (л)
   - Начальная крепость (%)
3. Нажмите "Рассчитать"
4. Получите объемы голов, тела и хвостов

### Скорость отбора

1. Нажмите на кнопку "⚡ Скорость отбора"
2. Введите:
   - Крепость (%)
   - Температура (°C)
   - Фактор оборудования (1.0 = стандарт)
3. Нажмите "Рассчитать"
4. Получите оптимальную скорость и рекомендации

### Температурные режимы

1. Нажмите на кнопку "🌡️ Температура"
2. Введите:
   - Крепость (%)
   - Атмосферное давление (гПа)
3. Нажмите "Рассчитать"
4. Получите температуру кипения воды и перегонки

## 🔧 Конфигурация

### База данных

Приложение использует SQLite для хранения данных. База данных создается автоматически при первом запуске в файле `vinokurnya.db`.

### Темы

Темапы переключаются через кнопку в заголовке окна приложения.
- **Темная тема**: Фон #1A1A2E, акцент #00FFAB
- **Светлая тема**: Фон #FFFFFF, акцент #008F6B
